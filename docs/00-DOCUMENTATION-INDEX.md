## Quick Reference

### Key Value Propositions
- **Real film data** - Presets derived from actual film behavior (proprietary cult files)
- **Predictable color response** based on empirical film measurements
- **Performance-first** architecture with Metal acceleration
- **Community-driven** recipe sharing ecosystem (QR-based, client-side)
